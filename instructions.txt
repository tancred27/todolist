Setting up the db (with phpmyadmin):
Create a user "root" with no password
Create a database called todolist and import the sql file

To install the required node modules and to start the app on localhost, run the following commands in cmd:
1) npm i
2) cd client
3) npm i
4) cd..
5) npm run dev
